# EventPulse

EventPulse is a web-based event management application developed as part of the WGU Software Engineering Capstone. It allows users to manage event information, search records, generate attendance reports, and securely access the application through user authentication.

## Features

* User authentication with Spring Security
* Create, edit, and delete events
* Search events by title or notes
* Attendance report generation
* Input validation
* Responsive Bootstrap user interface

## Technologies Used

* Java
* Spring Boot
* Spring Security
* Thymeleaf
* MySQL
* Bootstrap
* Maven
